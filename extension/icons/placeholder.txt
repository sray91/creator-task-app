# Extension Icons Placeholder

The extension needs icon files in this directory:
- icon16.png (16x16 pixels)
- icon32.png (32x32 pixels) 
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

For now, you can:
1. Create simple colored squares as PNG files
2. Use any icon creation tool
3. Or temporarily comment out the icons section in manifest.json

The extension will work without icons, but Chrome will show a default extension icon.
